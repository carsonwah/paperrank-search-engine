# PaperRank: A Reputation-based Search Engine for Academic Papers

## Getting Started

### Backend only

Install `python3`, `virtualenv`.

```bash
$ virtualenv env -p python3  # Use python 3.6+
$ source env/bin/activate
(env) $ pip install -r requirements.txt
(env) $ flask run
```

### Docker

Install `docker`.

```bash
$ docker-compose up
```
